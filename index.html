Submit an Issue on GitHub

How to Submit an Issue

To help us help you, please include the following information when submitting a bug report:

For Bug Reports

  • Description: A clear description of what the bug is
  • Steps to Reproduce: Step-by-step instructions to reproduce the issue
  • Expected Behavior: What you expected to happen
  • Actual Behavior: What actually happened
  • Device Information: Your device model and operating system version
  • App Version: The version of Fjáni you're using
  • Screenshots: If applicable, screenshots that help explain the problem

For Feature Requests

  • Feature Description: A clear description of the feature you'd like to see
  • Use Case: Explain how this feature would be useful to you
  • Additional Context: Any other information that might be helpful

Before Submitting

Before creating a new issue, please:

  • Check if the issue has already been reported by searching existing issues
  • Make sure you're using the latest version of the app
  • Try restarting the app to see if the issue persists
